Follow this discovery workflow to explore the schema:

1. Search for relevant modules using `discovery-search_modules`
2. For each interesting module, search data objects with `discovery-search_module_data_objects`
3. For each interesting table/view, inspect fields with `schema-type_fields`
4. Search for functions with `discovery-search_module_functions`
5. Check enum types with `schema-enum_values` when needed for filters
